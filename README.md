# Smart Street Light System

An AI-powered Smart Street Light System with Accident Detection and Emergency Response. The project combines Computer Vision, Machine Learning, and the MERN Stack to detect road accidents in real time, notify emergency services, and provide a web dashboard for monitoring incidents.

---

## Features

- Real-time accident detection using Computer Vision
- Multi-modal machine learning model utilizing camera and audio inputs
- Automatic emergency alert generation
- SMS notification for rapid emergency response
- MERN-based dashboard for monitoring incidents
- Responsive user interface
- Real-time incident reporting and management

## Project Overview

The system continuously analyzes live camera feeds and audio signals to identify road accidents. Once an accident is detected, the application immediately generates an alert, sends emergency notifications, and displays the incident on a web dashboard.

This helps reduce emergency response time and improves road safety through automated monitoring.

---

## Key Highlights

- Achieved approximately **90% accident detection accuracy** using a multi-modal machine learning model.
- Reduced incident reporting time to **under 10 seconds** using a MERN dashboard.
- Automated SMS notifications with an average **detection-to-alert latency of under 15 seconds**.

## Installation

Clone the repository

```bash
git clone https://github.com/Avani-14/Smart-Street-Light-System.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

### Machine Learning

```bash
cd ml-model
pip install -r requirements.txt
python app.py
```
